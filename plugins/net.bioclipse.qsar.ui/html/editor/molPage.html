QSAR Editor - Molecules Page

The Molecules Page of the QSAR Editor allows for adding molecular files to the dataset. The is done by drag and dropping a file in the viewer, or clicking the "Add" button, which brings up a wizard (see screenshot below).

Select a file and click Next to get to the Response Property selection. Here you can right-click on a property in the file and mark it as the response, which will import the values as response values into the QSAR dataset (see screenshot below).

When you are done, click "Finish". The Molecules Page shows the file in the list, and also indicates if any errors were encountered. Right-click to view or remove molecules with errors (see screenshot below).

The toolbar in the QSAR editor contains a shortcut icon in the top right corner to perform a complete build (calculate all selected descriptors for all selected molecules).
