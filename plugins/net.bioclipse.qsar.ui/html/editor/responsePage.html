QSAR Editor - Responses Page

The Responses Page of the QSAR Editor allows for entering and modifying response values for the chemical structures (see screenshot below). Click in the fields and make any changes. It is also possible to set the response unit by clicking in the corresponding entry. Note that you have to add the response unit first to the QSAR project on the Information Page before you can select it here.

The toolbar in the QSAR editor contains a shortcut icon in the top right corner to perform a complete build (calculate all selected descriptors for all selected molecules).
