QSAR Editor - Overview Page

The Overview Page of the QSAR Editor shows a summary of the active QSAR project. It is easy to see e.g. the number of chsmicals, descriptors, responses and to navigate to the corresponding page.

Some functionality is only available on the Overview page. For example it is here that you set AutoBuild to be ON or OFF. Read more about this on the QSAR Project page.

The toolbar in the QSAR editor contains a shortcut icon in the top right corner to perform a complete build (calculate all selected descriptors for all selected molecules).
