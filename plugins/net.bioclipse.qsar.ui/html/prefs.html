Bioclipse QSAR - Preferences

The Bioclipse preferences has a section for QSAR with three pages.

Descriptors: Here you can add additional descriptor definition files, to join with the Blue Obelisk Descriptor Ontology.

Providers: Here you can change the order of the descriptor providers being taken as default when adding a descriptor to the QSAR project.

Units: Here you can add additional Reponse Units, which later can be added to a dataset.
